# ocp-canton-sdk

High-level TypeScript SDK for Open Cap Table Protocol contracts on Canton Network.

See the [wiki](https://github.com/fairmint/ocp-canton-sdk/wiki) for full documentation. For the
documentation site, see [ocp-canton-sdk.fairmint.com](https://ocp-canton-sdk.fairmint.com/) (built from the [fairmint/web](https://github.com/fairmint/web) monorepo).
